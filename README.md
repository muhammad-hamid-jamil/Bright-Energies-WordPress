# Bright Energies – WordPress Website

Developed a B2B website for **Bright Energies**, an industrial coal supplier.  
Focused on product catalog and inquiry system for B2B clients.

## 🔧 Tech Stack
- WordPress
- Elementor Pro
- HTML, CSS, JavaScript
- RankMath SEO
- Google Tag Manager
